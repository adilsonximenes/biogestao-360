# Código de Conduta para Contribuições

## Nossa Missão

Este projeto tem como missão exclusivamente **acadêmica e educacional**. Qualquer contribuição deve estar alinhada com este propósito.

## Comportamento Aceito

- Contribuições para melhoria da precisão científica
- Correções de bugs
- Melhorias na documentação
- Sugestões para novos recursos educacionais

## Comportamento Não Aceito

- Qualquer tentativa de uso comercial do código
- Distribuição paga do software
- Remoção dos avisos de licenciamento
- Uso dos dados para fins lucrativos

## Consequências

Contribuições que violem este código de conduta serão rejeitadas e o contribuidor poderá ser banido do projeto.
