# Código de Conduta para Contribuições

## Nossa Missão

O **BioGestão 360** é uma ferramenta educacional gratuita para
organização de dados de saúde e atividade física. Contribuições
devem estar alinhadas com esse propósito.

## Comportamento Aceito

- Melhorias na precisão científica dos cálculos
- Correções de bugs e problemas de desempenho
- Melhorias na documentação
- Sugestões de novos recursos educacionais
- Adição de alimentos às tabelas nutricionais (TACO, IBGE, BioGestão 360)
- Melhorias na acessibilidade e usabilidade
- Traduções e adaptações regionais (outros estados/países)

## Comportamento Não Aceito

- Remoção dos avisos de licenciamento (CC BY-NC-ND 4.0)
- Tentativas de vender ou comercializar o software
- Uso dos dados dos usuários para fins não autorizados
- Modificação do sistema de autenticação para burlar controles de acesso
- Remoção dos avisos de segurança e orientações médicas/profissionais
- Alteração das referências científicas sem embasamento adequado
- Remoção das atribuições às bases de dados (Open Food Facts ODbL,
  TACO/UNICAMP, IBGE/POF)

## Sobre as Bases de Dados

O projeto utiliza três bases nutricionais com licenças distintas:

| Base | Licença | Obrigação |
|---|---|---|
| Open Food Facts | ODbL | Atribuição obrigatória |
| TACO/UNICAMP | Uso científico | Citação da fonte |
| IBGE/POF 2008-2009 | Uso público | Citação da fonte |

Contribuições que adicionem dados nutricionais devem respeitar
as licenças de origem e incluir a atribuição adequada.

## Sobre a Colaboração Voluntária

O projeto aceita colaboração voluntária dos usuários em troca
de acesso às seções exclusivas. Isso **não** configura venda
de software — o aplicativo em si é gratuito e open source.
Contribuições ao código devem respeitar essa filosofia.

## Responsabilidade Profissional

Este aplicativo é uma ferramenta de apoio educacional.
Contribuições que envolvam funcionalidades de saúde devem:

- Manter os avisos de que o app não substitui profissionais habilitados
- Preservar as referências às exigências do CREF (Educação Física)
  e CRN (Nutrição)
- Manter os alertas de segurança da seção Monte Seu Treino
  (SBV, Socorrismo Aquático, sinais de alerta)

## Consequências

Contribuições que violem este código serão rejeitadas e o
contribuidor poderá ser bloqueado do projeto.
